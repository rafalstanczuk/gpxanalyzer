# Side-by-Side Map & Chart View

Split-screen layout displaying map and elevation chart simultaneously. The left panel shows your GPS track on a detailed map with landmarks, roads, and terrain features clearly labeled. The right panel displays an elevation profile graph with color-coded zones representing different elevation ranges. A green pin marker on the map corresponds to the point highlighted in the elevation chart, demonstrating synchronized navigation between geographical and elevation data. Interactive controls allow zooming, panning, and detailed point selection.
