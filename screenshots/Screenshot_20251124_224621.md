# Elevation Profile with Gradient Analysis

Detailed elevation profile chart displaying altitude changes from 147m to 224m. Color-coded visualization shows green areas for ascents and red areas for descents. Interactive tooltip provides precise metrics: time, current elevation, gradient percentage with incline indicator, elevation gain, segment distance, and cumulative distance. The chart background features horizontal colored bands representing different elevation zones. Settings gear icon and navigation controls (pan, zoom in/out, statistics) available in the toolbar above the graph.
