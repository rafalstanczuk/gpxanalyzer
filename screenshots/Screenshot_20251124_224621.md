# Detailed Segment Analysis

In-depth segment analysis with automatic peak and valley detection. The elevation profile graph displays terrain changes with color-coded ascent (green) and descent (red) segments. A vertical dashed line marks the current point, with a detailed tooltip showing gradient percentage, elevation gain/loss, segment distance, and total activity distance. The app intelligently identifies and highlights important terrain features like steep inclines (shown with gradient indicators) and elevation changes throughout your route.
