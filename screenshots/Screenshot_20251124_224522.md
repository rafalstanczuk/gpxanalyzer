# Synchronized Map & Elevation View

Split-screen view combining map visualization with elevation profile. The map displays your GPS track with color-coded route segments, while the elevation chart below shows ascent (green) and descent (red) areas. Interactive tooltip displays detailed metrics including time, current altitude, elevation changes, and distance at any selected point. The green pin marker on the map corresponds to the point highlighted by the vertical dashed line on the elevation graph, demonstrating real-time synchronization between map and chart views.
