# File Management with Strava Sync

Activity file browser with dual-source support for local files and Strava integration. The interface features two tabs: "LOCAL" for device-stored GPX files and "STRAVA" for activities synced from your Strava account. Each activity card displays a map thumbnail with location pin, activity title, start location with full address, date and time, GPS coordinates, elevation, creator information, and file size. The "POWERED BY STRAVA" branding indicates secure API integration. Map thumbnails are automatically generated for quick visual identification of your activities.
