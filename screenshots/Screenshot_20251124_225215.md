# Activity File Browser with Strava Integration

File management interface with two source tabs: "LOCAL" (selected) for device files and "STRAVA" for synced activities. "POWERED BY STRAVA" branding displayed. Activity list shows cards with map thumbnails (grayscale OpenStreetMap previews) and green location pins. Each card displays: activity filename (e.g., "Morning_Trail_Run_16302360...", "Evening_Run_16285044393.g...", "test20230719.gpx"), full address location, date and time (2025-10-30, 2025-10-28, 2023-07-19), GPS coordinates (latitude/longitude), elevation in meters above sea level, creator ("Strava Sync." or "Sports Tracker"), author name, and file size in kilobytes (498 kB, 1112 kB, 229 kB, 586 kB). Map thumbnails help visually identify activity locations.
